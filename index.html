Эволюция подходов к реабилитации

Древний Египет – Рим (700 г. до н.э.)

Очищения, ритуальные практики. Римские пиршества с последующей рвотой – прототип булимии.

XIII–XVII века

Anorexia mirabilis – «святая анорексия». 181 случай религиозного поста. Екатерина Сиенская – классический пример.

1870-е

Уильям Галл и Эрнест Ласег описывают нервную анорексию как медицинское состояние. Начало медикализации.

1888

Комитет Клинического общества Лондона подтверждает связь щитовидной железы с психическими расстройствами.

Начало XX века

Моррис Симмондс: эндокринологическая модель анорексии. Лечение эндокринными препаратами.

1940–1954

Манфред Блейлер вводит понятие эндокринного психосиндрома. Монография «Эндокринологическая психиатрия».

1950-е

Интегративный этап: психоаналитическая модель + появление антипсихотиков. Развитие психоэндокринологии.

2007

Sonino & Fava: концепция реабилитации в клинической эндокринологии с психосоматическим подходом.

2020-е

Мультидисциплинарная модель. Телемедицина. Майндфулнесс-подходы. Интеграция доказательных практик.

Доказательные методы реабилитации

Когнитивно-поведенческая терапия (КПТ)

Золотой стандарт для булимии и BED. Выявление дисфункциональных убеждений, работа с образом тела, формирование адаптивных стратегий.

Высокая эффективность

Семейная терапия (FBT / Модсли)

Наиболее доказательный метод для подростков с анорексией. 90% хороших исходов при 5-летнем наблюдении. Активное вовлечение родителей в нормализацию питания.

Высокая эффективность (подростки)

Диалектическая поведенческая терапия (DBT)

Навыки осознанности, стрессоустойчивости, эмоциональной регуляции. Эффективна при РПП с эмоциональной дисрегуляцией.

Высокая эффективность

Мотивационное интервьюирование (МК)

Повышение приверженности лечению при ожирении и диабете. Наиболее эффективно в сочетании с КПТ.

Высокая эффективность

Майндфулнесс-подходы (MB-EAT)

86% исследований показали улучшение пищевого поведения. Снижение ИМТ, повышение самоэффективности при ожирении.

Высокая эффективность

Схема-терапия

Интегративный подход: работа с неудовлетворёнными детскими потребностями и паттернами, поддерживающими РПП. Экспериенциальные техники.

Умеренная доказательность

Арт-терапия

Вспомогательный метод: рисование, мандалы, песочная терапия. Работа с образом тела и эмоциональная регуляция.

Вспомогательный метод

Бариатрическая хирургия

Значительное улучшение при тревоге, депрессии и BED. Риск суицида возрастает в 1,9–3,8 раза — требуется мониторинг.

Высокая эффективность (с рисками)

Реабилитация по группам заболеваний

Нервная анорексия

Коды: F50.0 / 6B80 / AN (DSM-5)

Приоритеты:

  • Нутриционная реабилитация
  • FBT для подростков (90% успеха)
  • КПТ/DBT для взрослых
  • Соматическое восстановление

Нервная булимия

Коды: F50.2 / 6B81 / BN (DSM-5)

Приоритеты:

  • КПТ — золотой стандарт
  • Антидепрессанты
  • Нормализация питания
  • Работа с триггерами

Приступообразное переедание

Коды: F50.4 / 6B82 / BED (DSM-5)

Приоритеты:

  • КПТ, майндфулнесс (MB-EAT)
  • DBT
  • Стационарные программы
  • Структурированная среда

Ожирение с психиатрической коморбидностью

Код: E66

Приоритеты:

  • КПТ-О + МК + диетотерапия
  • Фармакотерапия (сибутрамин)
  • Бариатрическая хирургия
  • Мультидисциплинарная команда

Тиреоидная патология

Коды: F06.3, F06.4 / 6E62 (МКБ-11)

Приоритеты:

  • Коррекция гормонального статуса
  • 85% улучшаются при эутиреозе
  • КПТ при резидуальных симптомах
  • Бета-блокаторы при тиреотоксикозе

Сахарный диабет

Особенность: 39% имеют психические расстройства

Приоритеты:

  • Интеграция психотерапии в самоуправление
  • СИОЗС (серталин, флуоксетин)
  • КПТ + мотивационное интервьюирование
  • При шизофрении: низкий метаболический риск АП

Синдром Кушинга

Особенность: Избыток кортизола

Приоритеты:

  • Медикаментозная терапия
  • Заместительная гормональная терапия
  • Физиотерапия (массаж, ЛФК)
  • Психологическая поддержка

Патология паращитовидных желёз

Гиперпаратиреоз: депрессия 62%, тревога 53%

Приоритеты:

  • Коррекция кальция
  • Хирургическое лечение
  • Психофармакотерапия
  • КПТ при хронизации
